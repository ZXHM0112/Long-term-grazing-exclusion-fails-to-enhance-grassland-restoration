# Long term grazing exclusion fails to enhance grassland restoration

## Description of the data and file structure

The attachment contains the original data analysis and plotting R code, and also contains the original data. The code can be run directly as long as the relevant data is used in the run.

#### CSV File Name: plant-soil-microbial.csv

**Description of file:**

**Variables:**

*   Group: different treatments. EDG: extremely degraded grasslands; TGG: traditionally grazed grasslands being restored actively; SGE: short-term grazing exclusion in grasslands being restored actively; LGE: long-term grazing exclusion in grasslands being restored actively; NDG: non-degraded grasslands.
*   Time: sampling time
*   AGB: above-ground vegetation biomass,  unit is g m-2
*   BGB_0_30: below-ground vegetation biomass,  unit is g m-2
*   SOC: soil organic carbon
*   Richness.B: soil bacterial species richness
*   Richness.F:Â Soil fungal species richness
*   ---

  
#### CSV File Name: The plant community characteristics and soil properties.csv

**Description of file:**

**Variables:**

*   Group: different treatments. EDG: extremely degraded grasslands; TGG: traditionally grazed grasslands being restored actively; SGE: short-term grazing exclusion in grasslands being restored actively; LGE: long-term grazing exclusion in grasslands being restored actively; NDG: non-degraded grasslands.
*   Time: sampling time
*   Amino: Amino sugars of soil, unit is mg g-1
*   BNC: bacteria necromass carbon, unit is mg g-1
*   FNC: fungal necromass carbon, unit is mg g-1
*   MAOC: mineral-associated organic carbon, unit is g kg-1
*   POC: particulate organic carbon, unit is g kg-1
*   MAOC/SOC: the ratio of MAOC and SOC
   ---

  
#### CSV File Name: Characteristics of soil microbial community.csv
*   Group: different treatments. EDG: extremely degraded grasslands; TGG: traditionally grazed grasslands being restored actively; SGE: short-term grazing exclusion in grasslands being restored actively; LGE: long-term grazing exclusion in grasslands being restored actively; NDG: non-degraded grasslands.
*   Time: sampling time
*   complexity.b: network complexity of soil bacterial communities
*   complexity.f: network complexity of soil fungal communities
*   AVD.b: average variation degree of soil bacterial communities
*   AVD.f: average variation degree of soil fungal communities
*   MDS1.b: non-metric multidimensional scaling (NMDS)1 of soil bacteria
*   MDS1.f: non-metric multidimensional scaling (NMDS)1 of soil fungi
